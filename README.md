## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/davidrepository/biotech-app-recreation.git
cd biotech-app-recreation
```

### 2. Install dependencies

```bash
npm install
# or
yarn install
# or
pnpm install
```

### 3. Run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Visit App at:

```
http://localhost:3000
```
